# TasAR — Landing

Landing estática (HTML + Tailwind via CDN, sin build).

Fuente display: **Funnel Display 900** (Google Fonts).
Fuente cuerpo: **Inter**.
Datos numéricos: **JetBrains Mono**.

## Secciones

1. Ticker en vivo (USD/m² por barrio)
2. Hero — "El mercado inmobiliario, en datos."
3. Heatmap CABA con pins
4. Reporte de mayo (índice + curva)
5. Productos (4 cards: Tasador AI, Mapa de calor, Multi-tasador, Scraping)
6. Tasador interactivo (form + estimación)
7. Tasador AI conversacional (mock chat)
8. Consenso multi-tasador
9. CTA final — "Tomá decisiones con datos, no con intuición."

## Abrir

Doble click en `index.html`, o servir con:

```powershell
npx serve .
```
